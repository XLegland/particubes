# Number3

Number3 refers to a structure containing 3 number fields.
Number3 fields can be set individually or as a set of 3 values:

```
myNumber3.X = 42.0
myNumber3.Y = 12.3

myNumber3 = { X = 1, Y = 2, Z = 3 }
myNumber3 = { 1, 2, 3 }
```
